== Deleting nodes

[source,cypher,options="step"]
----
MATCH (gina:Human)
WHERE gina.name="Gina"
DELETE gina;
----

[options="step"]
This will not work because the node has relationships!


== Deleting relationships

[source,cypher,options="step"]
----
MATCH (gina:Human)-[r]-()
WHERE gina.name="Gina"
DELETE r, gina;
----

== Clearing the database

[source,cypher,options="step"]
----
MATCH (n)
OPTIONAL MATCH (n)-[r]->()
DELETE n, r;
----
