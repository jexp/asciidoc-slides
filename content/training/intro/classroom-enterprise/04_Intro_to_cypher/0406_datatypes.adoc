== Datatypes

[options="step"]
- decimal: double or integer (hexadecimal and octal as well)
- string
- boolean
- lists
- maps


== Operators

[options="step"]
- mathematical: `+`, `-`, `*`, `/`, `%`, `^`
- comparison: `=`, `<>`, `<`, `>`, `<=`, `>=`, `IS NULL`, `IS NOT NULL`
- boolean: `AND`, `OR`, `XOR`, `NOT`
- string concatenation: `+`
- collections: `+`, `IN`
