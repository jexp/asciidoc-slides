== From SQL to Cypher

[options="step"]
`MATCH` allows you to find a pattern in the graph

[options="step"]
[source,cypher]
----
MATCH (h:Human)<-[:EMPLOYEE]-(d:Dept)
RETURN h.name, d.name
----

[options="step"]
`WHERE` allows you to filter the results

[options="step"]
[source,cypher]
----
MATCH (h:Human)<-[:EMPLOYEE]-(d:Dept)
WHERE d.name = "IT-Department"
RETURN h.name, d.name
----
